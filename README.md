# myapp

A new Flutter project.

## Getting Started
Diseña una aplicación móvil en Flutter para una app de clases de baile llamada “DanceAshley”.

La app debe tener un estilo moderno, elegante y minimalista con una paleta de colores negro, rosa pastel y blanco.

Pantallas requeridas:

1. Welcome Screen:
- Logo o imagen de una bailarina (desde internet o estilo ilustración)
- Título: “DanceAshley”
- Texto: “Aprende a bailar desde casa con los mejores estilos”
- Dos botones grandes:
  - Login (con degradado negro a rosa)
  - Sign Up (outline)

2. Login Screen:
- Título: “Login”
- Campos:
  - Email
  - Password
- Botón principal con degradado negro a rosa
- Texto: “¿No tienes cuenta? Registrarse”

3. Sign Up Screen:
- Título: “Sign Up”
- Campos:
  - Nombre
  - Email
  - Password
- Botón principal elegante

Diseño:
- Bordes redondeados
- Sombras suaves
- Inputs blancos con esquinas redondeadas
- Botones grandes y modernos
- Uso de imágenes desde GitHub o internet
- Espaciado limpio y profesional

Objetivo:
Que se vea como una app real lista para producción, con estilo femenino, moderno y atractivo.
